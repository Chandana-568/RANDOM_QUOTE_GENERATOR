# 🎯 Random Quote Generator

A simple and interactive **Random Quote Generator** built using **HTML, CSS, and JavaScript**. Every click displays a new inspirational quote, making it a fun project to practice DOM manipulation, event handling, and working with js arrays.


## ✨ Features

- 🎲 Generate random quotes instantly
- 💡 Clean and responsive UI
- ⚡ Fast and lightweight
- 📱 Mobile-friendly design
- 🎨 Modern styling using CSS
- 🔄 New quote on every button click

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)
